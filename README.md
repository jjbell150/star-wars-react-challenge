## Setup

### Installing dependencies

```bash
npm install
```

### Building

```bash
npm run build
```

## Running

### Server

```bash
npm run start
```

### Development mode

```bash
npm run start:dev
```

### Preview mode

```bash
npm run start:preview
```

## Testing

```bash
npm run test
```

## Linting

### Check for issues

```bash
npm run lint
```

### Fix issues

```bash
npm run lint:fix
```
